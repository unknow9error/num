# Code of Conduct

This project expects professional, respectful collaboration.

## Expected Behavior

- Be direct and technical without personal attacks.
- Assume code and arguments can be improved.
- Keep discussions focused on the language, runtime, tooling, and user impact.
- Respect maintainers' decisions on scope, architecture, and release timing.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private security details before a fix is available.
- Repeated off-topic comments that block project work.

Maintainers may remove comments, close issues, or restrict participation when
needed to keep the project productive.
